# Problem Statement - Sentiment analysis for Gujarati is underexplored due to limited resources. This project aims to develop and evaluate a sentiment classification system for Gujarati text by comparing traditional machine-learning approaches with multilingual transformer-based models in a low-resource setting.

This project studies text classification under low-resource conditions using Gujarati movie reviews.

## Motivation
Gujarati is underrepresented in NLP research despite its large speaker base.

## Approach
We compare a classical TF-IDF baseline with multilingual transformer models to evaluate data-efficient learning.

## Models
- TF-IDF + Logistic Regression
- Multilingual Transformer (XLM-R / DistilBERT)

## Results
Transformer models outperform classical baselines under low-resource constraints.

## Notes
Training was performed under limited computational resources.
